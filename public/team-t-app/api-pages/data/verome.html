verome api

曲を検索して、歌詞まで見る。

Verome API で YouTube Music 由来の楽曲を検索し、選んだ曲の歌詞(LRCLib経由)を表示します。

Base URL: https://verome-api.deno.dev
/api/search, /api/lyrics
No auth
表示内容
曲名・アーティスト・サムネイルの検索結果一覧と、選択した曲の歌詞テキストをまとめて表示します。
キーワードを入力して検索してください。
-
-
楽曲データは実際の検索結果に基づきます。歌詞は LRCLib のデータベースに一致した場合のみ取得できます。API: Verome-API (GitHub)
