anime quotes api

アニメの一言を、すぐ貼れる形で。

Animechan のランダム引用を取得して、作品名・キャラ名・本文を読みやすく表示します。API は無料枠で動き、失敗時はローカルのサンプルへ切り替えます。

Base URL: https://api.animechan.io/v1
Rate limit: 5 requests/hour
Auth: optional supporter tier
