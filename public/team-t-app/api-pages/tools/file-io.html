Disposable file handoff / file.io

使い捨ての共有リンクを、
その場で作って渡す。

file.io は、1回ダウンロードされたら消える一時共有サービスです。ファイルだけでなく短文も送れるので、一時的なパスワードやメモの受け渡しにも使えます。

1回で消える共有ダウンロード後は自動削除。期限も追加できます。
ファイル/短文の両対応ファイルを選ぶか、短いテキストを直接送信できます。

共有リンクを作る

ファイルを選ぶか、短文を入力してアップロードします。

no login auto delete REST API
ファイルか短文を送信すると、ここに共有リンクが表示されます。

返却データ

成功時は共有キー、リンク、期限を表示します。

見どころ

  • ファイルはダウンロード1回で削除される
  • 短文でも一時共有リンクに変換できる
  • 期限を付けて自動削除を早められる

APIドキュメント

https://www.file.io/

参考コマンド

curl -F "file=@test.txt" https://file.io/?expires=1w curl --data "text=temporary note" https://file.io
送信に失敗した場合は、CORS 制約や一時的な API 停止を疑ってください。 official site
