JSON storage / Pantry

JSON をクラウドのバスケットへ、
軽く保存して取り出す。

Pantry は、API またはダッシュボードから CRUD できる JSON ストレージです。小さな設定値、テストデータ、簡易メモなどをすばやく保存する用途に向いています。

軽量な JSON 保管バスケット単位で読み書きします。
小規模プロジェクト向けデータベースを立てずに試せます。

バスケットを読む / 書く

Pantry ID とバスケット名を入力し、JSON を GET / PUT で扱います。

JSON CRUD basket small data
Pantry ID を入れると、保存済み JSON の読み書きができます。

返却データ

読み込み時は取得した JSON、保存時は送信内容を表示します。

見どころ

  • バスケット単位で JSON を保持できる
  • 保存前に JSON をローカルで編集できる
  • 小規模な設定値や共有メモに使いやすい
