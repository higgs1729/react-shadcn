Public upload API / FileUp

期限と閲覧回数を決めて、
一時ファイルを配る。

FileUp は公開アップロード API を持つ軽量なファイル共有サービスです。アップロード時に有効期限と最大閲覧回数を指定でき、受け取ったURLをそのまま共有できます。

公開アップロードログイン不要で REST API から送信できます。
制御しやすい共有期限分数と閲覧回数を数値で渡せます。

ファイルをアップロードする

ファイルを選んで、期限と閲覧回数を指定して送信します。

no login expire control view limit
ファイルを選んでアップロードすると、返却URLがここに表示されます。

返却情報

成功時はファイル名、共有URL、期限、閲覧回数を表示します。

見どころ

  • 公開 API でそのまま送れる
  • 期限と閲覧回数を細かく調整できる
  • 返却 JSON をその場で確認できる
