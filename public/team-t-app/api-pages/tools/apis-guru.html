OpenAPI catalog / APIs.guru

公開API定義を、
ひとつの検索窓でたどる。

APIs.guru は公開APIの OpenAPI 定義カタログです。プロバイダ名やサービス名で検索し、登録件数と代表APIを見つける用途に向いています。

定義カタログ公開APIの件数や名前を横断検索できます。
検索結果を絞るサービス名入力で候補を即座に絞り込みます。

検索する

API名やキーワードを入れて、公開カタログを絞り込みます。

no keycatalogsearch
キーワードを入れると、catalog からAPI名を検索します。

返却データ

代表的なAPI名と、その情報ページを一覧表示します。

見どころ

  • 公開APIの一覧を横断検索できる
  • サービス名の候補集めに向く
  • API定義ページへの導線を持つ
