File hosting / URL shortening / 0x0.st

ファイルも URL も、
ひとまず短い共有リンクにする。

0x0.st は使い捨てのファイル共有サービスです。ファイルを上げるだけでなく、長い URL を短縮リンクに変換する用途にも使えます。

ワンステップ共有アップロードしたら、そのまま短い URL を受け取れます。
ファイル / URL 対応タブで送信対象を切り替えられます。

送信モードを切り替える

ファイルを送るか、URL を短縮します。返却はどちらも共有リンクです。

temp share file shorten
モードを選んで、リンクを作成してください。

返却データ

成功時はテキストで共有 URL が返る想定です。

見どころ

  • ファイル共有と URL 短縮を1ページにまとめた
  • 返却がテキストでも扱いやすい
  • 失敗時はそのままエラー文を表示する
