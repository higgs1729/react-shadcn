compiler / CodeX

オンラインコンパイラの公開情報を、
送信前のひな形として整理。

CodeX は多言語対応のオンラインコンパイラです。現時点で安定した公開 endpoint を検証できなかったため、コード送信の雛形と参照リンクをまとめるページにしています。

送信準備言語とコードを整えて実行前に確認します。
代替手順公開 endpoint が戻ったときにそのまま切替可能です。

コードひな形を作る

言語とコードを入力して、送信コマンドを作ります。

compilersnippetplaceholder
CodeX 用の送信コマンドを生成します。

返却データ

公開情報と、送信に使うコマンドを表示します。
